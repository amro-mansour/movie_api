Documentation Page

The objective of this api is to build the server side component of a movies web application. The web application will provide users information about different movies, actors, genres, directors, etc. Users will also be allowed to sign up, add and update their personal information, and create a list of their favourite movies.

These are the instructions on how to use the API

Logic URL HTTP Method Query Parameters Request Data Body Format Response Data Body Format
Return a list of ALL movies to the user /movies GET None None JSON data type containing details about all the movies
Return data (description, genre, director, image URL, whether it’s featured or not) about a single movie by title to the user /movies/:title GET None None JSON object containing details of specific movie
Return data about a genre (description) by name/title (e.g., “Thriller”) /movies/genre/:genreName GET None None Json object containg information about a specific movie genre
Return data about a director (bio, birth year, death year) by name /movies/directors/:directorName GET None None JSON object containing informatio about a specific director (bio, birth year, death year)
Allow new users to register /users POST None JSON object is added, e.g. { "name": "Mark", "dateOfBirth": "dd/mm/yyyy", "favouriteMovies": [movie1, movie2, movie3] } JSON object sent back from the request plus a random ID number assigned to the user
Allow users to update their user info (username) /users/:id PUT None JSON object containing the data to modify, e.g. {"name": "New Name"} Json object sent back from the request with updated infromation
Allow users to add a movie to their list of favorites (showing only a text that a movie has been added) /users/:id/:movieTitle POST None JSON object containing info about the movie the user wants to add, e.g. {"title": "name of movie", "director": "name", "genre": "movie genre"} A message reporting the movie has been added succefully
Allow users to remove a movie from their list of favorites /users/:id/:movieTitle DELETE None None A message reporting that the movie has been succefully removed
Allow existing users to deregister /users/:id DELETE None None A message reporting that the user has been succefully removed
